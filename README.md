# Student Performance Dashboard

## Overview
This Power BI dashboard analyzes student academic performance, attendance, and behavior.

## Features
- Data Modeling
- DAX Measures
- Interactive Dashboard
- Attendance Analysis
- Subject-wise Performance
- Behavior Analysis
- Drillthrough
- Slicers
- Conditional Formatting

## Dataset
- Students.csv
- Scores.csv
- Attendance.csv
- Behavior.csv

## KPIs
- Total Students
- Average Score
- Attendance %
- Behavior Count

## Visualizations
- Bar Chart
- Line Chart
- Donut Chart
- Table
- Cards

## Tools Used
- Power BI Desktop
- DAX
- Power Query
